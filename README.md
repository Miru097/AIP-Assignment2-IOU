### How to start

`$ git clone`

`$ npm install`

`$ cd client`

`$ npm install`

`$ cd ..`

`$ npm run dev`

### MongoDB connect

If you want to use MongoDB Cloud Atlas, use the link in `\config\keys.js` to replace the `mongoURI` in `\config\default.json`. 

If you want to change back to Community Server, just replace it again.
